# Synent Technologies Internship - Task 5

## Project Name
To-Do List Web App

## Description
A responsive and interactive To-Do List Web Application built using HTML, CSS, and JavaScript.  
The application allows users to add, complete, and delete tasks with localStorage support so tasks remain saved even after refreshing the page.

---

## Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- localStorage support
- Responsive design
- Modern user interface
- Smooth animations and hover effects

---

## Technologies Used
- HTML5
- CSS3
- JavaScript
